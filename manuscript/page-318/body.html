And this is the reason why all things should have a price set upon them, because thus there may be exchange at any time, and if exchange then dealing. So money, like a measure, making all things commensurable equalises them: for if there was not exchange there would not have been dealing, nor exchange if there were not equality, nor equality if there were not the capacity of being commensurate: it is impossible that things so greatly different should be really commensurate, but we can approximate sufficiently for all practical purposes in reference to Demand.
